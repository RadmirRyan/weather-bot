# Название проекта
Wheater Telegram Bot
## Описание
Данный бот подскажет погоду в городе любому пользователю Telegram. 
В коде есть ошибки, простите, я только учусь.

Смена города (/reset)
и многое другое...
## Технологии
- Python 3.10
- Библиотека telebot
- API wttr.in
## Как запустить
1. `git clone https://...`
2. `pip install -r requirements.txt`
3. `python main.py`
